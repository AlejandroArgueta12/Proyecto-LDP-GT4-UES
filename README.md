# Gestor de Proyectos Simple - Ciclo 1/2026

## Descripción del Proyecto
Este proyecto consiste en una aplicación de interfaz de línea de comandos (CLI) desarrollada para la asignatura **Lógica de Programación Orientada a Objetos**. El sistema permite la gestión de empleados y proyectos, la creación de tareas y el registro de horas trabajadas con validaciones de seguridad.

## Autor
*Alejandro Argueta** - Estudiante de Ingeniería en Desarrollo de Software.

## Entrega #1: Avance en Pseudocódigo
Para esta primera etapa, se presenta una solución funcional desarrollada en **PSeInt** que incluye:
* **Módulo de Registro:** Alta de empleados y proyectos.
* **Módulo de Tareas:** Creación y asignación de actividades.
* **Módulo de Control de Tiempos:** Validación de jornada laboral real (máximo 24 horas diarias) y cálculo de totales.

## Cómo ejecutar el avance
1. Clonar este repositorio.
2. Abrir el archivo `GestorProyectos.psc` en **PSeInt**.
3. Asegurarse de utilizar el perfil **Flexible**.
4. Ejecutar el programa y seguir las instrucciones del menú interactivo.

## Tecnologías utilizadas
